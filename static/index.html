QtBot - Extensible Discord Bot

https://github.com/r3dFIVE/QtBot

An interactive Discord Chat Bot written from the ground-up using Qt5.

  • Discord (https://discord.com/) Chat Bot written from ground-up using Qt5.
  • Expose custom API to JavaScript engine. Can interact with Discord's extensive API,
    scrape websites, interact with databases, etc.
  • Scripts can be added/reloaded to provide additional commands and functionality without
    the need to restart/recompile bot.
  • Multiple database engines supported: MongoDB, PostgreSQL, MSSQL Server, MariaDB/MySQL,
    SQLite.
  • Custom MongoDB driver using the Mongo C/C++ bindings.
